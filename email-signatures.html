Adding signatures to Gmail

  1. Click Copy on the signature you want
  2. Open Gmail → Settings → See all settings
  3. Scroll to Signature, click + Create new
  4. Press Ctrl+A (Cmd+A) then Ctrl+V (Cmd+V) to paste
  5. Save changes

Tip: If formatting is lost, use right-click → "Paste as rich text."
