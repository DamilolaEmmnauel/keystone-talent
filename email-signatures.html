How to add these signatures in Gmail

  1. Click the Copy HTML button below each signature
  2. Open Gmail → Settings (⚙️) → See all settings
  3. Scroll to the Signature section
  4. Click + Create new, name it (e.g. "Keystone Talent")
  5. In the signature editor, press Ctrl+A (or Cmd+A) then Ctrl+V (Cmd+V) to paste
  6. Save changes at the bottom of the page

Note: If pasting loses formatting, paste into the Gmail signature editor using right-click → "Paste as rich text" or "Paste and match style."
